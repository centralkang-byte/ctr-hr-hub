# seed-dev.ts 설계 문서

**날짜**: 2026-03-04
**목적**: 전체 메뉴 테스트를 위한 개발용 시드 데이터 생성
**대상 법인**: CTR-KR (한국), CTR-CN (중국) 중심

---

## 배경

- `prisma/seed.ts` (마스터 시드): 회사/역할/권한/부서/직급 등 기준 데이터만 포함, 테스트 계정 4개뿐
- 메뉴별 실제 테스트에 필요한 트랜잭션 데이터 부재
- **seed.ts 오류**: `prisma generate` 누락으로 발생 → 수정 완료

## 실행 방법

```bash
# 마스터 시드 (기준 데이터)
npx prisma db seed

# 개발용 시드 (트랜잭션 데이터)
npm run seed:dev
```

---

## 설계 원칙

1. **멱등성**: `deterministicUUID` + upsert 사용, 재실행 안전
2. **의존성**: 기존 seed.ts 실행 후 실행 (company/dept/grade/policy 참조)
3. **분리**: `prisma/seed-dev.ts` 별도 파일
4. **현실성**: 실제 HR 시나리오에 맞는 데이터 분포
