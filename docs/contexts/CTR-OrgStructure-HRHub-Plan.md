> **기준일자:** 2026-03-01 (조직도 기준)
> **작성일:** 2026-03-20
> **목적:** 실제 CTR 그룹 조직 구조를 HR Hub에 반영하기 위한 GAP 분석 + 실행 계획

## 1. 의사결정 요약

| 항목 | 결정 |
|------|------|
| 배포 범위 | **전 계열사 동시 배포** (~2,500명) |
| 매트릭스 보고 | **첫 배포부터 필수** (Solid + Dotted Line, 평가 가중치 분리) |

## 6. 매트릭스 보고 구조 — 시스템 설계 필요사항

### 현재 시스템
- Position에 `reportsToPositionId` (단일 Solid Line만 지원)
- Performance에 매트릭스 평가 가중치 개념 없음

### 필요한 변경

#### 6-1. Position 모델 확장
```prisma
model Position {
  // ... 기존 필드
  reportsToPositionId    String?   // Solid Line (기존)
  secondaryReportsToId   String?   // Dotted Line (신규)
  secondaryReportWeight  Int?      // Dotted Line 평가 가중치 % (e.g., 30)
  // primaryReportWeight = 100 - secondaryReportWeight
}
```

#### 6-2. 실제 매트릭스 사례 (조직도 기반)
| 대상 | Solid Line (주) | Dotted Line (부) | 가중치 |
|------|----------------|-----------------|--------|
| 대합공장 팀장들 | CTR모빌리티 대표 (양현철) | OE 사업부문장 (엄용일) | 70:30 |
| CTR CHINA 본부장들 | CTR CHINA 총경리 (리밍) | CTR 그룹 주무 본부장 | 법인:그룹 |
| CTR AMERICA 팀장들 | CTR AMERICA 법인장 (김주형) | CTR 그룹 주무 본부장 | 법인:그룹 |
| CTR VINA 팀장들 | CTR VINA COO (홍영수) | AM 사업부문장 (이성훈) | 법인:사업부문 |

#### 6-3. 겸직 (Concurrent Position) 지원
한 Employee가 여러 법인에서 active EmployeeAssignment를 가질 수 있어야 함.

**겸직 사례:**
- 이동옥: CTR 대표 + CFO 겸직
- 정병주: CTR 품질경영팀장 + CTR모빌리티 국내법인 겸직
- 이경수: 모빌리티 경영관리팀장 + EHS팀장 + 정보보안팀장
- 박양원: AM R&D센터장 + 설계팀V 팀장
- 방우영: SCM본부장 + OM팀 팀장 겸직
- 한성욱: 재무회계팀장 (CTR + CTR에코포징 겸직)
- 최연식: 피플앤컬처팀장 (CTR + 전 계열사 그룹 주무)

**시스템 설계:**
```
Employee (1명)
├── EmployeeAssignment #1: CTR, Position=대표이사, isPrimary=true
├── EmployeeAssignment #2: CTR, Position=CFO, isPrimary=false
└── EmployeeAssignment #3: CTR-MOB, Position=품질경영팀장(겸), isPrimary=false
```
`isPrimary` 필드 추가 + 급여는 primary assignment 법인에서 지급.
