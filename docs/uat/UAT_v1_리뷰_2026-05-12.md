## 2. UAT 피드백 결과 엑셀 리뷰

### 2.1 골격 평가

| 시트 | 평가 |
|------|------|
| 리포트 작성법 | 그대로 유효 (심각도 4단계 + 작성 예시 + 5개 규칙) |
| 피드백 로그 | 헤더 18 컬럼 OK. **100 rows 빈 슬롯**(FB-001~FB-100) — UAT 2주 동안 100건 입력 슬롯은 충분 |
| 일별 트리아지 | Day 1-10 × 9 columns 빈 템플릿. **수식 없음** — 수동 입력 전제 |
| 모듈별 집계 | 14개 모듈 (인사관리/출퇴근/휴가관리/급여/결재승인/마스터데이터설정/성과관리/채용/보상/분석/온보딩오프보딩/규정준수/My Space/기타) — **휴직 누락** |
| 심각도별 집계 | 4 × 9 빈 템플릿. **수식 없음** |
| UAT 판정 | 통과 기준 6 + 최종 판정 + 사인오프 — 그대로 유효 |

### 2.2 🔴 핵심 결함 — 자동 집계 수식이 없음

피드백 로그에 100건이 입력되어도 **모듈별 집계 / 심각도별 집계 / 일별 트리아지**가 자동으로 채워지지 않습니다 (모두 빈 셀, 수식 없음).

**권장 수식** (v2 작성 시 적용):

```
[모듈별 집계]
B2: =COUNTIFS(피드백로그.B:B,A2,피드백로그.G:G,"Critical")
C2: =COUNTIFS(피드백로그.B:B,A2,피드백로그.G:G,"Major")
D2: =COUNTIFS(피드백로그.B:B,A2,피드백로그.G:G,"Minor")
E2: =COUNTIFS(피드백로그.B:B,A2,피드백로그.G:G,"개선")
F2: =SUM(B2:E2)
G2: =IFERROR(COUNTIFS(피드백로그.B:B,A2,피드백로그.K:K,"해결")/F2,0)

[심각도별 집계]
B2: =COUNTIFS(피드백로그.G:G,A2,피드백로그.K:K,"신규")
... (확인중/수정중/수정완료/재테스트/종료/보류 × 4 심각도)

[일별 트리아지]
B2 (신규): =COUNTIFS(피드백로그.J:J,DATE(2026,4,21))  -- 발견일 = Day 1
C2 (미해결 누적): =COUNTIFS(피드백로그.J:J,"<="&DATE(2026,4,21),피드백로그.K:K,"<>해결")
```
