## 공통 레이아웃 프레임

모든 대시보드 페이지가 공유하는 상단 구조:

```
┌─────────────────────────────────────────────────────────┐
│  PageHeader: "{대시보드 이름}"                            │
│  Breadcrumb: 홈 > 인사이트 > {대시보드 이름}              │
│                                                          │
│  ┌─ 필터 바 (공통) ────────────────────────────────────┐ │
│  │ 법인: [그룹 합산 ▼]   기간: [최근 12개월 ▼]   부서: [전체 ▼] │ │
│  └─────────────────────────────────────────────────────┘ │
│                                                          │
│  ┌─ AI Insight 배너 (G-2에서 연결) ────────────────────┐ │
│  │  ✨ "이번 달 기술연구소 이직률이 전사 평균의 2.2배…"    │ │
│  │     → [이직 분석에서 상세 보기]                       │ │
│  └─────────────────────────────────────────────────────┘ │
│                                                          │
│  ┌─ KPI 카드 행 ──────────────────────────────────────┐ │
│  │  (대시보드별 3~5개 KPI 카드)                         │ │
│  └─────────────────────────────────────────────────────┘ │
│                                                          │
│  ┌─ 차트 영역 ────────────────────────────────────────┐ │
│  │  (대시보드별 다름 — 아래 개별 정의)                   │ │
│  └─────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

**공통 컴포넌트:**
- `AnalyticsFilterBar` — 법인/기간/부서 드롭다운 (CompanySelector 재활용)
- `AiInsightBanner` — G-1에서는 placeholder, G-2에서 실제 AI 텍스트 연결
- `KpiCard` — 숫자 + 라벨 + 전월 대비 △ 표시 + 아이콘
- `ChartCard` — recharts 차트를 감싸는 카드 (제목 + 로딩 스켈레톤 + 에러 바운더리)
- `WidgetSkeleton` — 기존 CRAFTUI Phase 3 패턴 재활용

**색상:**
- Primary 차트: `#5E81F4`
- Secondary 시리즈: `#8B5CF6`, `#10B981`, `#F59E0B`, `#EF4444`, `#06B6D4`
- 배경: `#F5F5FA`, 카드: `#FFFFFF`, 텍스트: `#1C1D21`

### 🚨 기간 필터 기본값: TTM (Trailing 12 Months)

> **"1월의 저주" 방어** — 2026년 1~2월에 접속하면 "올해" 기준으로는 점 1~2개짜리 차트만 보임. 경영진이 이걸 보면 즉시 신뢰를 잃음.

**기본값:** 최근 12개월 (TTM) — 2026년 2월 접속 시 2025년 3월~2026년 2월 데이터 표시

**기간 드롭다운 옵션:**

| 옵션 | 설명 |
|------|------|
| **최근 12개월 (기본)** | now - 12months ~ now (TTM) |
| 최근 6개월 | now - 6months ~ now |
| 최근 3개월 | now - 3months ~ now |
| 2026년 | 1월~12월 (YTD if 진행 중) |
| 2025년 | 1월~12월 |
| 사용자 지정 | 시작일~종료일 직접 선택 |

### 공통 Empty State 원칙

모든 대시보드 페이지에서 데이터가 0건일 때:
- 차트 영역: "데이터가 없습니다" 안내 텍스트 + 연한 아이콘
- KPI 카드: "—" 또는 "0" 표시 (NaN/undefined 절대 금지)
- 에러 발생 시: "데이터를 불러올 수 없습니다. 잠시 후 다시 시도해주세요." + 재시도 버튼

### 팀 건강 점수 (상단 히어로)

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│              🟡 72점 / 100                               │
│         [반원형 게이지 차트]                               │
```

**점수 계산 (5지표 가중 평균):**

| 지표 | 가중치 | 0점 기준 | 100점 기준 |
|------|:------:|---------|----------|
| 초과근무 | 20% | 주 10h+ 초과 | 주 0h 초과 |
| 연차 사용률 | 15% | 30% 이하 | 80% 이상 |
| 성과 등급 분포 | 20% | 가이드 대비 편향 심함 | 가이드 일치 |
| 이직 위험 | 25% | 팀의 30%+ 고위험 | 0명 고위험 |
| 번아웃 위험 | 20% | 팀의 20%+ 위험 | 0명 위험 |

**해석:** 90+ 🟢 건강 / 70-89 🟡 주의 / 50-69 🟠 경고 / 0-49 🔴 위험

### 팀원 상태 테이블

```
┌─────────────────────────────────────────────────────────┐
│  이름    │ 초과근무 │ 연차사용 │ 성과 │ 이직위험 │ 상태    │
│  김대리  │ 8h/주   │ 60%   │ M+   │ LOW    │ 🟢     │
│  박사원  │ 15h/주  │ 20%   │ B    │ HIGH   │ 🔴     │
│  이과장  │ 5h/주   │ 80%   │ M    │ MEDIUM │ 🟡     │
│  정주임  │ 12h/주  │ 45%   │ M    │ LOW    │ 🟡     │
└─────────────────────────────────────────────────────────┘
```

**매니저 제한:**
- Position.reportsToPositionId 기반 direct report만 표시
- 이직 위험 "수준"(HIGH/MEDIUM/LOW)만, 점수 수치는 미노출
- 급여/Compa-Ratio 데이터 접근 불가

### 🚨 팀원 0명 Empty State (Gemini Patch)

신임 매니저, 발령 직후, 또는 조직도 데이터 미연결 시 팀원 0명일 수 있음. 이 경우 건강 점수 계산에서 division by zero → NaN → UI 붕괴 방지 필수.

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│         👥 조회할 팀원 데이터가 없습니다 (0명)              │
│                                                          │
│    현재 직속 팀원이 배정되지 않았거나,                      │
│    조직도 데이터가 아직 연결되지 않았습니다.                 │
│                                                          │
│    인사팀에 문의하시거나 조직도 업데이트를 확인해주세요.      │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

**방어 코드:** 팀원 0명이면 게이지/테이블/추천 액션 전부 숨기고 위 Empty State만 표시. `score = 0 / 0` 같은 연산 자체를 하지 않음.
