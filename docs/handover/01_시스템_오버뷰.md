## 5. 데이터 흐름 (요약)

```
[Browser]
    ↓ NextAuth + Azure AD (SSO)
[Next.js Middleware]  → session.role JWT pin (SUPER_ADMIN cross-company bypass)
    ↓
[App Router Page or API Route]
    ↓ withRLS (companyId 격리) + withPermission (RBAC 16×8×5)
[Prisma Client]
    ↓ DATABASE_URL (Supabase Direct Connection 5432)
[PostgreSQL]
    ↓ Trigger (도메인 이벤트 27 handlers)
[Event Bus] → 알림 (Teams / SES / Web Push) + Nudge engine 11 rules
```

상세 구조는 [docs/ARCHITECTURE.md](../ARCHITECTURE.md).

## 8. Sessions 168~217 동안 추가된 신규 기능

UAT v1 작성 시점(2026-04 초) 이후 추가된 기능 12+종. 본 시스템의 최근 1~2개월 변경 분량은 인계 시 별도 학습 필요:

### V2 홈 (4 역할별)

`/home`에서 역할에 따라 다른 홈 화면:
- `HrAdminHomeV2`: 결재 inbox 요약, 온보딩/오프보딩 tracker, 휴가 통계, recognition feed
- `ManagerHomeV2`: 팀원 현황, 결재 대기, 1:1 일정, 팀 성과 요약
- `ExecutiveHomeV2`: HR KPI + 채용 퍼널 + 인력 트렌드
- `EmployeeHomeV2`: 내 할일, 휴가 잔액, 목표 진행률, 인정 피드, 출퇴근 위젯

### 보상 (Compensation) 대규모 확장

- **Off-Cycle Comp 3 발의 경로**: 매니저 제안 / HR 주도 / 자동 트리거 — ApprovalFlow 다단계 + 급여 역전 방어 + Self-approval Skip + 미래 시행일 cron
- **Compensation Letter PDF**: `compensation-letter-pdf.ts` 생성 → S3 저장 → 이메일 배치 발송 → 버전 관리
- **Total Rewards Statement** (`/my/total-rewards`): 기본급/상여/수당/복리후생/포상 연간 집계 + 도넛 차트 + 트렌드
- **PayBandChart**: 직급별 연봉 밴드 시각화, 4곳 적용 (ProfileSidebar / CompensationTab / SimulationTab / CompReview)

### 성과 (Performance) 확장

- **GoalRevision**: 목표 버저닝 + 배치 수정 + QGP(분기 목표 진행) 데이터 오염 방어
- **QuarterlyReview**: 6 API + 3 UI 페이지 (My / Admin / Detail) + 직원·매니저 듀얼 제출 + 마스킹
- **캘리브레이션 DnD 배치 조정**: @dnd-kit 드래그·드롭 + 다중선택 + 배치 API
- **AI 자기평가 도우미**: Claude 코멘트 초안 생성
- **성장 여정 시각화**: MyResult 과거 사이클 추이 라인 차트

### LOA (휴직) Phase 1-3

- 6-state 워크플로 (REQUESTED→APPROVED→ACTIVE→RETURN_REQUESTED→COMPLETED/REJECTED)
