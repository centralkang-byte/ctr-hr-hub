## P01. 마스터-디테일 레이아웃

```
┌─────────────────────────────────────────┐
│ 페이지 헤더 (제목 + 액션 버튼)              │
├──────────────┬──────────────────────────┤
│ 마스터(좌)     │ 디테일(우)                 │
│ w-80~96      │ flex-1                   │
│ 리스트/트리    │ 탭 네비게이션               │
│ 검색+필터     │ 콘텐츠 영역                │
│ 선택 하이라이트│ 하단 액션 바               │
└──────────────┴──────────────────────────┘
```

**규칙:**
- 좌측: `w-80` or `w-96`, `border-r border-[#E8E8E8]`, 독립 스크롤
- 선택 항목: `bg-[#E8F5E9] border-l-2 border-[#00C853]`
- 모바일: 목록 → 클릭 시 풀스크린 전환

## P03. 데이터 테이블 + 필터 바

```
┌─────────────────────────────────────────┐
│ [검색🔍] [필터▼] [필터▼]    [+추가] [⋮]  │
│ 활성: [Product팀✕] [여성✕] [재직중✕]      │
├─────────────────────────────────────────┤
│ ☐│이름↕│부서│직급│상태│액션               │
│ ☐│김상우│인사│과장│🟢재직│⋮               │
├─────────────────────────────────────────┤
│ ◀ 1 2 3 ▶                10건/페이지▼   │
└─────────────────────────────────────────┘
```

**규칙:**
- 검색: 좌측 `w-64`, 디바운스 300ms
- 필터 칩: `bg-[#FAFAFA] rounded-full px-3 py-1 text-xs border border-[#E8E8E8]` + ✕
- 정렬: 클릭 토글 `↕→↑→↓→↕`
- 체크박스: 선택 시 벌크 액션 바 나타남
- 빈 상태: 일러스트 + "결과 없음" + 필터 초기화

- tooltip: 섹션 이름 표시

### 역할별 필터링

```typescript
// navigation config에서 visibleTo 배열로 관리
// SUPER_ADMIN은 모든 섹션 접근
// 빈 섹션(items 0개)은 자동 숨김
// canAccessModule 기존 함수 재활용
```

### 반응형

| 브레이크포인트 | 동작 |
|-------------|------|
| < md | 사이드바 숨김 + 햄버거 메뉴 (오버레이 드로어) |
| md~lg | 접힌 상태 (w-16) 기본 |
| > lg | 펼친 상태 (w-64) 기본 |

### 상태 저장
- 펼침/접힘: `localStorage('sidebar-collapsed')`
- 섹션 아코디언 열림/닫힘: `localStorage('sidebar-sections')`
- 페이지 이동 시 현재 활성 섹션 자동 펼침

### NP02. 캘리브레이션/9-Grid

```
     높음│ C(2) │ A(5) │
  성과   │ D(1) │ B(8) │
     낮음│ E(0) │ C(3) │
```

### NP04. 전자서명 플로우

- 모바일 최적화 필수
